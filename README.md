# To-Do App con React y Node.js

Aplicación web para la gestión de tareas y metas personales, desarrollada con React en el frontend y Node.js con Express en el backend. Permite registrar, visualizar y eliminar información mediante una arquitectura cliente-servidor.

---

## Descripción del proyecto

El objetivo de la aplicación es brindar un espacio donde el usuario pueda organizar sus tareas y metas, registrando datos como nombre, descripción y fecha. El frontend se encarga de la interfaz de usuario, mientras que el backend proporciona una API REST para el manejo de la información.
